# 06 - Spicy Mode Guardrails

## Goal

Keep spicy mode funny and viral without crossing into hostility.

## Guardrails

- Punch up at behavior patterns, not personal worth
- Avoid insults tied to identity, intelligence, or dignity
- Avoid profanity in default share outputs
- Keep jokes short and remixable

## Copy Style

Good:

- "You delegate like a leader and revise like a hawk."
- "You want creativity, but in a very specific lane."

Avoid:

- "You're impossible to work with."
- "You are toxic."

## Confidence-Aware Spice

When score confidence is low or mixed:

- reduce intensity
- use softer language like "sometimes" and "can feel"

When confidence is high:

- allow slightly sharper phrasing
- keep tone playful, not punitive
