# 07 - Zero-Friction First Run

## First-Run Principle

No signup wall. No profile setup. No tutorial maze.

## v1 Flow

1. User lands on homepage
2. Picks agent type (`Coding Agent` or `Chatbot`)
3. Copies provided prompt/markdown
4. Pastes agent output
5. Sees results immediately
6. Optional: save/share/retake

## Friction Rules

- No authentication before first result
- No required email capture before share card generation
- Ask for account creation only after value is delivered

## Growth Loops

- "Try another agent" CTA on result page
- "Compare with friend" share line
- Auto-generated card text for fast posting
