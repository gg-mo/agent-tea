# 08 - Visual Identity and Character Figures

## Goal

Create a recognizable illustration system for result pages and share cards, inspired by test-character ecosystems but with original style and palette.

## Art Direction (v1)

- Cohesive, vector-friendly character family
- Distinct pose/emotion by type archetype
- Bright but non-MBTI palette
- Clean silhouette for small-card readability

## System Components

- 16 base character variants (one per type)
- 4 color families mapped to dominant dimension mood
- Expression overlays for normal vs intrusive-thought mode
- Share-card-safe crop variants

## Production Approach

- Start with 4 anchor archetypes
- Derive the rest through controlled variations
- Maintain one design language to avoid looking random

## Technical Output Targets

- Web: SVG or high-res PNG
- Share cards: 1:1 and 4:5 exports
- Consistent background motif for recognizability in feeds

## Why It Matters

Visual identity drives recall and repost behavior. A recognizable character system can become the product signature.
