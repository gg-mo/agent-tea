# 04 - Social Proof: Most Common and Rarest

## v1 Feature

Display:

- "Most common type this week"
- "Rarest type this week"

## UI Placement

- On result page below primary type
- Optional mini-module on landing page

## Data Logic

- Rolling 7-day window
- Count completed tests only
- Minimum sample threshold before showing labels

If sample size is too small:

- show "Collecting signal..." instead of potentially misleading stats

## Viral Effect

- Users compare themselves to the crowd
- Rarity encourages shares
- Common-type label creates group identity behavior
