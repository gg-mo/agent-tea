# 10 - Name Shortlist

## Recommendation

**Best pick: `Agent Tea`**

Why:

- Short, funny, and social-native
- Implies opinion/gossip in a playful way
- Matches the "agent judging human" hook without sounding harsh
- More viral-feeling for consumer launch positioning

## Strong Alternatives

- `HumanGrade`
- `AgentMirror`
- `HowMyAISeesMe`
- `Promptality`
- `HumanType`

## More Playful Alternatives

- `ReadMeRobot`
- `VibeCheckByAI`
- `MyAIsOpinion`
- `Type Me, Bot`

## Name Direction Notes

- Avoid names that sound overly corporate or clinical
- Avoid names that are too harsh (feels like punishment)
- Favor names that can sit on a share card naturally

## Landing Tagline Pairing (for Agent Tea)

- Headline: "Your AI has tea about you."
- Subhead: "Get your agent-eye personality type in 2 minutes."
